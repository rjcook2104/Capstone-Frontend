# SuperSecure AI - Store Division Manager Portal
**Capstone Stage 1: Requirements & Prototyping**

## 🚀 Overview
An AI-integrated security dashboard designed to detect workplace tension and facilitate employee de-escalation training.

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS v4
- **Routing:** React Router DOM
- **State Management:** React Hooks (useState, useNavigate)

## 📍 Key Features
- **Role-Based Access:** Distinct portals for Managers (Ops Center) and Employees (Learning).
- **Smart Matrix:** 6-camera live grid with "Priority Focus" on tension alerts.
- **Hardware Inventory:** Real-time battery and assignment tracking for body cams.
- **Learning Portal:** Interactive de-escalation quizzes based on flagged incidents.

## 🏃 How to Run
1. `npm install`
2. `npm run dev`
